# questionApp_front
A simple app to post questions and answers. This is the front end implementation served on an Express server.

Currently includes:

1. Sign up.
2. Login.
3. Add question.
4. Add answer.

Typescript + React + Sass + Webpack

to run:

1. npm install.
2. typings install.
3. npm run server.
